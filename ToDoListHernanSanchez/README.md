# Aplicación de Lista de Tareas

Esta es una aplicación de consola para la gestión de una lista de tareas, 
desarrollada en C# utilizando .NET Framework 4.8.

## Instrucciones para compilar y ejecutar la aplicación

1. Clona el repositorio:
git clone  https://github.com/HernanGoat77/PruebaQ10.git

2. Abre el proyecto en Visual Studio

3. Compila el proyecto

4. Ejecuta la aplicación

## Funcionalidades
- Agregar tareas con descripción y fecha límite(opcional).
- Listar todas las tareas.
- Marcar tareas como completadas.
- Eliminar tareas de la lista.
